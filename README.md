# 📝 To-Do List Manager

Вебзастосунок для керування завданнями, реалізований на HTML5, CSS3 та JavaScript ES6+.

## 🚀 Функціонал

- ✅ Додавання завдань (кнопка або Enter)
- ✏️ Редагування тексту завдання
- 🗑️ Видалення завдання з підтвердженням
- ☑️ Позначення завдання як виконаного
- 🔍 Фільтрація: Всі / Активні / Виконані
- 🧹 Пакетне видалення виконаних завдань
- 💾 Автозбереження у `localStorage`
- 🔢 Лічильник активних завдань
- 🛡️ Захист від XSS
- 📱 Адаптивний дизайн (mobile-friendly)

## 📁 Структура проєкту

```
todo-list-manager/
├── index.html   — структура інтерфейсу (HTML5)
├── style.css    — стилі та адаптивність (CSS3)
├── app.js       — вся логіка застосунку (JS ES6+)
├── schema.sql   — схема бази даних MySQL (серверний варіант)
└── README.md
```

## ▶️ Запуск

### Варіант 1 — через live-server (рекомендовано)
```bash
npm install -g live-server
cd todo-list-manager
live-server
```
Відкриється `http://127.0.0.1:8080`

### Варіант 2 — через Python
```bash
python -m http.server 8080
```

### Варіант 3 — напряму
Відкрити `index.html` у браузері (подвійний клік).

## 🗄️ База даних (MySQL)

Для серверного варіанту виконайте:
```bash
mysql -u root -p < schema.sql
```
